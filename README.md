# GISM
GISM is the abbreviation of general industrial smart monitoring.
It is designed to complete the task of intelligent detection with YOLO V4.

## Installation

* build docker image

```bash
docker build -t gism:vx.x .
```

* load the docker image

```bash
sh install_gism.sh
```

* run gism

```bash
sh run_gism.sh
```
